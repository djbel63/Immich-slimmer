# 🚀 Immich-Slimmer

A lightweight tool designed to automatically optimize and downscale your Immich library, enabling massive disk space savings on a scheduled basis.

## ✨ Features
- **Automatic Resizing**: Shrinks heavy photos to your preferred dimensions (e.g., 1920px).
- **Smart Threshold**: Configurable minimum size (`MIN_SIZE`) to only process files that actually take up space.
- **Auto-Refresh**: Automatically notifies Immich to reflect file size changes in the UI immediately.
- **Data Integrity**: Preserves all original EXIF metadata (Date, GPS, Camera info).
